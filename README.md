# Usage
<!-- usage -->
```sh-session
$ npm install -g nuxt-coreui-cli
$ nuxt-coreui-cli COMMAND
running command...
$ nuxt-coreui-cli (-v|--version|version)
nuxt-coreui-cli/0.0.0 win32-x64 node-v12.14.1
$ nuxt-coreui-cli --help [COMMAND]
USAGE
  $ nuxt-coreui-cli COMMAND
...
```
